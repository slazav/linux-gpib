ibrsp

Name

ibrsp -- conduct serial poll (device)

Synopsis

#include <gpib/ib.h>

int ibrsp(int ud, char *result);

Description

ibrsp() serial polls the device specified by ud. The status byte is stored in the location specified by result.

Return value

The value of ibsta is returned.
