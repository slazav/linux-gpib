ibwrta

Name

ibwrta -- write data bytes asynchronously (board or device)

Synopsis

#include <gpib/ib.h>

int ibwrta(int ud, const void *buffer, long num_bytes);

Description

ibwrta() is similar to ibwrt() except it operates asynchronously. ibwrta() does not wait for the sending of the data bytes to complete, but rather returns immediately.

While an asynchronous operation is in progress, most library functions will fail with an EOIP error. In order to sucessfully complete an asynchronous operation, you must call ibwait() and until the CMPL bit is set ibsta. Asynchronous operations may also be aborted with an ibstop() or ibonl() call.

Return value

The value of ibsta is returned.
